# [NPC Dedekinto: Criação de Lac 100](/pt-br/knowledge-bases/21/articles/49458-npc-dedekinto-criacao-de-lac-100)

Agora será possível compor o item “Poeira de Lac 100” no NPC “Dedekinto”. Para isso serão necessários os itens abaixo:

[10] Poeira de Lac
[01] Shiny Laktorerium pieces

-> Em caso de falha, todos os itens serão destruídos.
-> Em caso de sucesso, uma Poeira de Lac 100 imóvel será criada.
