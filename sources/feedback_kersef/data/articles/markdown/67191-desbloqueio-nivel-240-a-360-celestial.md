# [Desbloqueio nível 240 a 360 - Celestial](/pt-br/knowledge-bases/21/articles/67191-desbloqueio-nivel-240-a-360-celestial)

Ao atingir os níveis 240, 280, 320 e 360 do Personagem Celestial o jogador deve utilizar o item “Proteção Divina” para realizar o desbloqueio dos mesmos.

![Image](https://lh5.googleusercontent.com/5n4OsaRUOBWqpkLWV2cO5STXP0JVvuLEiJ6v3OQpgwfKxI6BdknwC3ooHjbV-mlM1aVrLQNBbuod5OmWjSrZvGGhxZbulYOm4Q_6HxFjz3oow06ud_TYtADB_OEC8zP2PXs-DDd25aJTMuNrp0yINuw)

No level 240 o jogador não precisa estar com ambos os personagens no mesmo level para desbloquear o level 240, basta apenas um dos personagens alcançar o level 240 para desbloquear.

A partir do level 280 o jogador precisa estar com ambos os personagens no mesmo level para utilizar o item, ao utilizar o level será desbloqueado em ambos os personagens.

O item é composto pelo NPC Dedekinto localizado na Cidade do Kefra.

![Image](https://lh4.googleusercontent.com/nVsVv1s0wSQb3iI1lXrVJMBxeRGCZvFgq4EvBwlBG-JXbX_EsEr6aNBMP9nOMYRu8o-wQ-75IFj-O7YZySaY8675jIKg-Dtyq-dlUSl_e4cw7-QJQGpBmjjM-4_8hSLXSIjMaaQb6lcv5KXuyQs7RFw)

Os itens necessários para composição são: Coração Beriel Amald (2) e Alma Beriel (2).

**![Image](https://lh3.googleusercontent.com/_UkwE8XCeE366RxiQP-aA8mjxGtKCvo9usL42bANDWQHmNS1bRu3d-gqjGhQTo_LpT2ABdCNO4lD-UExkbcpD79GQWoTfaaA50i6bKwGEvQa_H8lYnPulgTFUEnBmf7HOulrMD1Vd_nYaZyffSLfPpM)**

**Observações:**

* A chance de sucesso da composição é de 25%.
* Não existe falha na utilização do item "Proteção Divina" no desbloqueio do level 240.
* A partir do level 280 existe uma chance de falha de 50%.
* Necessário fazer o desbloqueio em apenas um dos personagens.
* A composição só pode ser realizada com personagem Celestial level 200 ou superior.
