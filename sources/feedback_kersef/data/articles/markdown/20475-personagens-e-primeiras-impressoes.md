# [PERSONAGENS E PRIMEIRAS IMPRESSÕES](/pt-br/knowledge-bases/21/articles/20475-personagens-e-primeiras-impressoes)

|  |  |  |  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| |  |  | | --- | --- | | Classe | Características | | Transknight | Transknight que utiliza os ataques físicos e tem uma grande quantidade de vida. Muito eficiente não somente na função de caçar os monstros, mas também nos PvPs. | | Foema | São os personagens que utiliza os ataques mágicos normalmente como sua principal força ofensiva, mas utilizando a combinação certa de armas e skills, poderá transformá-la numa guerreira do mesmo nível dos outros personagens. | | Beastmaster | O Beastmaster pode ser criado de diversas formas através das diversas skills de natureza que possui, o Beastmaster pode evocar os monstros mais fortes que o si próprio ou ganhar um bônus no próprio físico se transformando num híbrido de homem e bicho. | | Huntress | A agilidade da Huntress permite dominar os campos de batalha é muito útil, principalmente nos casos de PvP ou mesmo nas caças de monstros. A facilidade de utilizar as armas de longa distância e a habilidade de acessar o banco faz dela uma das caçadoras mais adaptadas ao combate. | |
|  |
| Uma das características do WYD é que existem muitas linhagens de skills para cada personagem, podendo assim apenas aprender as skills desejados ou virar uma especialista numa determinada linhagem aprendendo todas as skills dessa linhagem. |

A primeira cidade que conhecerá é a cidade de Armia, a cidade natal de todos os iniciantes do WYD. Ao criar um personagem, o jogo será iniciado dentro da cidade de Armia.

![Image](https://web.archive.org/web/20101025095036im_/http://imagem.ongame.com.br/wyd/introducao/guia_noob/img/first_guidetitle1_06.gif)

![Image](https://web.archive.org/web/20101025095036im_/http://imagem.ongame.com.br/wyd/introducao/guia_noob/img/first_guideimg_31.gif)

Os movimentos do personagem são controlados basicamente através do mouse.

Coloque o cursor do mouse em cima do local que deseja se movimentar e clique com o botão esquerdo do mouse.

Caso queira movimentar a visão do jogo, basta apertar e segurar o botão de scroll do mouse e movimentar para a direção desejada.
Poderá também mudar a visão rapidamente apertando a tecla “Tab”.

|  |
| --- |
| Dentro da cidade de Armia existe um local chamado de Campo de Treinamento, onde um personagem de lvl1 pode treinar tranquilamente para depois, no campo de Armia, encontrar monstros mais desafiadores.    Deixe o cursor do mouse em cima do monstro que deseja atacar e clique com o botão esquerdo do mouse, assim o seu personagem iniciará o ataque contra o monstro selecionado.  Se pressionar a tecla 'T' ou clicar no ícone de auto ataque da tela, o modo ataque automático será ativado e o personagem atacará o alvo selecionado até que derrote o monstro.    ![Image](https://web.archive.org/web/20101025095036im_/http://imagem.ongame.com.br/wyd/introducao/guia_noob/img/first_guideimg_33.gif) |
