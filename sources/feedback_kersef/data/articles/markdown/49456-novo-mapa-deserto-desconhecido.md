# [Novo Mapa: Deserto Desconhecido](/pt-br/knowledge-bases/21/articles/49456-novo-mapa-deserto-desconhecido)

O novo mapa do deserto desconhecido poderá ser acessado no NPC “Angela Sister” localizado em Noatun. O acesso poderá ser feito apenas no canal novato e será necessário entregar o item “Artefato Ancestral”.

-> O mapa será reiniciado a cada 20 minutos.
-> O item “Artefato Ancestral x10” poderá ser composto utilizando os itens “Pergaminho Selado x1” + “Poeira de Lac x20” + “Selo Mágico”.
-> Novo item: “Shiny Laktorerium pieces” poderá ser obtido no mapa “Deserto Desconhecido" e pode ser utilizado para compor o item “Poeira de Lac 100”.
-> Novo item: ”Dragon Soul” poderá ser obtido no mapa “Deserto Desconhecido” e pode ser utilizado para compor o item que dá acesso ao novo mapa “Dragon Field”.

**Novas Atualizações:**

**-> Aumentado o drop do item Dragon Soul no monstro BoneDragon
-> Aumentado o drop do item Fragmento de Alma no monstro Scorpion
-> Aumentado o drop do item Poeira de Lac no monstro Fire Gagoil**
